>>> UNIT: L1.1 What is a model?
>>> COMPONENT: TEXT
TITLE: Reading
BODY:
A *model* maps inputs (called _features_) to outputs (called _labels_). *Training* fits that mapping to historical data; _inference_ is using the trained model on new data.

Key terms:
- *feature* — an input variable the model reads
- *label* — the value the model predicts
- *inference* — running the trained model on new inputs
<<<
